## TITLE OF THE PROJECT.
NAMMA RAILU BUDDY
## DESCRIPTION
Namma Railu Buddy is an android application designed as a passenger guide to city workers who rely on passanger trains. The project aims to use generative AI and community driven data to solve common navigation and information issues at small railway stations
## FEATURES
- AUTOMATIC ALARM TRIGGERING WHEN THE DESTINATION IS 5KM AWAY
- PLATFORM PING
- DISTANCE CALCULATION BETWEEN TWO STATIONS
- AUTOMATIC ALARM TUROFF AFTER 10 SECONDS
## TECHNOLOGIES USED
- KOTLIN
- JETPACK COMPOSE
- FIREBASE CONSOLE
- LOCATION GEOFENCING
- ANDROID STUDIO
